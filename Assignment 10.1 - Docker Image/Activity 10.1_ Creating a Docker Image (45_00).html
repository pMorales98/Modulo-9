Learning Outcome Addressed:

  1. Create a Docker image.

In this activity, you will be using Visual Studio (VS) Code to create an image for a web application that displays “Hello World - my first Docker Image!”. Make sure to install the Docker extension for VS Code:

Prior to beginning this activity, be sure to review the submission instructions below to ensure that you collect the required screenshots as you progress through the activity.

To complete the activity, perform the following steps:

  1. Download the following starter file to your local machine: Assignment 10.1 - Docker Image.zip. The zip file contains the initial code for this activity.
  2. Open the folder in Visual Studio Code:

  3. Add a file called “Dockerfile” and copy the following contents to the file:

    FROM node:12.16.3

    WORKDIR /code

    ENV PORT 80

    COPY package.json /code/package.json

    RUN npm install

    COPY . /code/

    CMD [ "node", "src/server.js"]

    Make sure to add the Dockerfile file in the folder “Assignment 10.1 - Docker Image” and to save it after editing it. The Dockerfile contains instructions for the Docker engine to create the Docker image. For more information regarding the contents of the Dockerfile, refer to this video from the 2020 DockerCon: How to Get Started with Docker Links to an external site..

  4. Now you are ready to start creating an image of this web application. You will need to go to the command line to run the Docker commands to create an image. In Visual Studio Code, navigate to Terminal. Then, select New Terminal:

  5. Run the following command from the prompt to create an image:

    Note: When you run this command, you should be in the same folder where you downloaded the Assignment 10.1 - Docker Image.zip starter file. This should also be the same folder where you created the Dockerfile. The space at the end of the docker build command (“.”) means “current directory”.

    docker build --tag hello-world.

  6. You should now have created a Docker image for the Hello World application. To view the Docker image, run the following command from the prompt:

    docker images

  7. Now you can run your image. Execute the following command from the prompt:

    docker run -p 8080:80 --name hello -d hello-world

    Note: For a detailed description of each Docker command, including explanations of each of the options, you can run --help for each of the commands. See the following example:

    docker run --help

  8. Next, open your browser and navigate to the URL of http://localhost:8080/.

    Your “Hello World - my first Docker Image!” page will be displayed.

Congratulations! You created your first Docker image, and you also ran the image from inside the Docker container.

Submission Instructions:

Your submission for this activity should be a Word document that includes the following screenshots, each labeled for the step that the screenshot represents:

  1. Dockerfile added to the “Activity 10.1 - Docker Image” folder.
  2. Screenshot of running the docker build command.
  3. Screenshot of running the docker images command.
  4. Screenshot of running the docker run command to run the image in the container.
  5. Screenshot of a web browser to show that you have navigated to the URL of http://localhost:8080/. The web page should display “Hello World - my first Docker Image!”
